LimaDetector Class

Device Properties
Property name
Property type
Description
DetectorDescription Tango::DEV_STRING A simple user free text intented to identify the Detector.
DetectorIP Tango::DEV_STRING Ip Address of the Detector
DetectorType Tango::DEV_STRING Define the type of the connected Detector . i.e: Basler Simulator
FileFormat Tango::DEV_STRING Define the format of image files : - EDF - NXS - RAW
FilePrefix Tango::DEV_STRING Define the prefix used for the image files name.
FileIndexFormat Tango::DEV_STRING Define the format of the index used for image files names.
FileNbFrames Tango::DEV_SHORT Define the number of frames per file.
FileTemporaryPath Tango::DEV_STRING Define the root path for temporary generated files.
[default = .]
FileTargetPath Tango::DEV_STRING Define the Path where Files will be generated, only when savingFile is checked.
[default = ./data]
DebugModules Array of string Define Lima Modules for which we need some traces in the console.
DebugLevels Array of string Define Lima verbose level of traces in the console.
